# Smart-Road-Monitoring-IoTSystem-Prototype
This project is a Smart Road monitoring prototype developed as an individual academic project.
The system utilizes IoT technology to monitor road conditions and display data through a web-based interface.
The project focuses on sensor data acquisition, microcontroller integration, and basic web monitoring.

## Features
- Road condition monitoring using sensors
- Data transmission from ESP32 to web application
- Web-based monitoring interface
- Basic data visualization

## Technologies Used
- ESP32
- IoT Sensors
- Arduino IDE
- HTML
- CSS
- JavaScript
- Visual Studio Code

## System Overview
The ESP32 collects data from sensors installed on the road prototype.
The collected data is processed and sent to a web interface for monitoring purposes.
